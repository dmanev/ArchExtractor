1. Make X90_QAC.prj read-write
2. Open X90_QAC.prj using text editor and check that all compiled components are included
3. Create empty folder 'temp'
4. Ensure that project is placed in D:\X90_BCM\DEVELOPMENT\X90_BCM\K0\L7\Build\QAC
5. Double-click on X90_QAC.prj (to open it using QAC tool)
6. Launch 'Analyse project with CMA'
7. Check that all files are analyzed successfully
8. Open 'File Metrics' and choose STTLN metrics
9. Open 'File'->'Export Data' and set file name for metrics export
